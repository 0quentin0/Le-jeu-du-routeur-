# Le jeu du routeur

This is a python project for an event in the [French Mindustry Discord](https://discord.com/invite/hzGPWhZSGV).
You are playing a snake your goal is to survive the wawe of crawler.

## Installation
1. clone the repository
2. install the dependencies
3. run `routeur.py` using python

## Dependencies
- [pygame](https://www.pygame.org/news)
